KeePass Online

This little service allows you to browse passwords from your KeePass database stored online on one of the cloud storages.

Security

KeePass Online takes security extremly seriously and that's why all the work with the KeePass database happens in your browser only. Neither your master password nor database content travels across the wire to our or other server. That's the best security anybody can provide.

The whole project is open source at GitHub, so anyone can check that it doesn't contain any malicious code.

Limitations

KeePass Online supports only the newest format of the database - KDBX files. Also, only databases secured by master passwords are supported. Please vote here for Keyfile support.
